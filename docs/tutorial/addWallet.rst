Add Wallet
==========

=================
About
=================

This page provides the details on how the options available for adding a wallet to Verto.

.. image:: img/createWallet/createVertoWallet.png
   :alt: Create Verto Wallet
   :align: center

=================
Overview
=================

When creating a wallet, you a provided with a few options. However, for the most part, the adding of a wallet is broken down into two main categories.

--------------------------
Create A New Account
--------------------------

You will need to create a new account in the event that you do not yet have one. For now, the wallet only supports creating Ledger accounts. These are the accounts we use for the Qualified Investors. However, we are looking into creating EOS accounts in the near future.

.. image:: img/createWallet/createLedger.png
   :alt: Create Verto Wallet
   :align: center

Read The Tutorial For Creating A Ledger Account:

.. toctree::
   :maxdepth: 1

   addwallets/createLedger.rst

--------------------------
Adding Existing Account
--------------------------

You are able to add both Ledger as well as EOS accounts to the wallet.

Note that you will require:

* A public key for Ledger accounts
* A private key for EOS accounts
